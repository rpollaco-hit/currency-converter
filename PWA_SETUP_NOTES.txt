PWA setup files for Currency Converter

1. Add these files to the project root:
   - manifest.json
   - service-worker.js

2. Add these lines inside <head> in index.html:
   <link rel="manifest" href="./manifest.json">
   <meta name="theme-color" content="#7C3AED">
   <link rel="apple-touch-icon" href="./assets/icon-192.png">

3. Add this before </body> in index.html:
   <script>
     if ("serviceWorker" in navigator) {
       window.addEventListener("load", () => {
         navigator.serviceWorker.register("./service-worker.js");
       });
     }
   </script>

4. Put these icons in assets:
   - icon-192.png
   - icon-512.png

5. Commit and push:
   git add manifest.json service-worker.js index.html assets/icon-192.png assets/icon-512.png
   git commit -m "Add PWA support"
   git push origin main
